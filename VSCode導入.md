## VS Code導入

### この章の位置づけ

この章では、Visual Studio Code（以下、VS Code）をインストールし、ローカルにある業務フォルダのファイルを自分で確認・編集する方法を扱います。

プログラム開発の知識は必要ありません。次の基本操作ができるようになることが目標です。

- ファイルの内容を確認・編集する
- Markdownファイルをプレビューする
- ファイルやフォルダを新しく作る
- 必要なファイルや文章を検索する

### VS Codeとは

VS Codeは、Microsoftが提供している無料のエディターです。Windows、macOS、Linuxに対応しています。

一般的なメモ帳と違い、左側にフォルダとファイルの一覧、中央に編集画面を表示できます。そのため、複数のファイルをフォルダ単位で整理しながら編集する用途にも向いています。

この章では、主にMarkdownファイル（拡張子が `.md` のファイル）を扱います。Markdownは、見出し、箇条書き、表などを簡単な記号で表現できるテキスト形式です。

### VS Codeをインストールする

1. VS Code公式ダウンロードページを開く
   - https://code.visualstudio.com/download
2. 自分のOSに合うものをダウンロードする
3. OSごとの手順でインストールする

### VS Codeを日本語化する

VS Codeをインストールした直後は、メニューやボタンが英語で表示される場合があります。最初にMicrosoft公式の日本語言語パックをインストールし、画面を日本語化します。

この項目だけは、日本語化する前の画面に合わせて英語のボタン名を記載しています。

1. VS Codeを開く
2. 画面左端の `Extensions` アイコンを押す
3. 検索欄に `Japanese Language Pack` と入力する
4. `Japanese Language Pack for Visual Studio Code` を選ぶ
5. 提供元が `Microsoft` であることを確認する
6. `Install` を押す
7. `Change Language and Restart` が表示されたら押す
8. VS Codeが再起動し、メニューが日本語になったことを確認する

日本語言語パックは、次の公式ページから開くこともできます。

- https://marketplace.visualstudio.com/items?itemName=MS-CEINTL.vscode-language-pack-ja

日本語に切り替わらない場合は、次の手順を行います。

1. macOSでは `Command + Shift + P`、WindowsとLinuxでは `Ctrl + Shift + P` を押す
2. 表示されたコマンド パレットに `Configure Display Language` と入力する
3. `Configure Display Language` を選ぶ
4. 言語一覧から `日本語（ja）` を選ぶ
5. VS Codeを再起動する

ここから先は、日本語化された画面を前提に説明します。

### 画面の見方

VS Codeを開くと、主に次の領域が表示されます。

| 場所 | 名前 | 役割 |
| --- | --- | --- |
| 画面の左端 | アクティビティ バー | ファイル一覧、検索、拡張機能などの機能を切り替える |
| アクティビティ バーの右側 | サイド バー | フォルダーやファイルの一覧、検索結果などを表示する |
| 画面の中央 | エディター | ファイルの内容を表示・編集する |
| 画面の下部 | ステータス バー | ファイル形式、文字コード、行番号などの状態を表示する |

### 業務フォルダを開く

VS Codeでは、作業に使うフォルダを最初に開きます。

1. VS Codeを開く
2. 上部メニューの `ファイル` を押す
3. `フォルダーを開く...` を選ぶ
4. `Claude業務フォルダ` を選ぶ
5. `開く` を押す
6. 左側のエクスプローラーにフォルダーとファイルの一覧が表示されることを確認する

ワークスペースの信頼を確認する画面が表示された場合は、自分で作成したフォルダーであることを確認してから信頼します。入手元が不明なフォルダーは、内容を確認せずに信頼しないでください。

### ファイルを開く

1. 左側のエクスプローラーで、開きたいフォルダーの左にある矢印を押す
2. 開きたいファイル名を押す
3. 中央のエディターに内容が表示されることを確認する

ファイル名を1回押すと一時的なプレビューとして開き、別のファイルを押したときに同じタブへ切り替わることがあります。ファイルを開いたままにしたい場合は、ファイル名をダブルクリックします。

### ファイルを編集・保存する

ここでは、既存のMarkdownファイルを編集します。

1. 左側のエクスプローラーで `10_業務ルール` を開く
2. `営業メール作成ルール.md` を開く
3. 中央のエディターで、必要な文章を編集する
4. 内容を確認して保存する

未保存の変更があるファイルは、タブのファイル名の右側に丸印が表示されます。保存すると丸印が消えます。

| パソコン | 保存 |
| --- | --- |
| macOS | `Command + S` |
| Windows、Linux | `Ctrl + S` |

上部メニューの `ファイル` から `自動保存` を有効にすると、変更内容を自動保存できます。自分で確認してから保存したい場合は、自動保存を無効のまま使います。

### なぜMarkdownで書くのか

Markdownは、文章に簡単な記号を付けて、見出し、箇条書き、表などを表現する書き方です。

Markdownには、次のような利点があります。

- シンプルな形式で、編集や検索がしやすい
- 文章の構造が明確になり、人にもAIにも内容が伝わりやすい
- 特定の文書作成ソフトに依存せず、さまざまなツールで扱える

プログラミングの知識がなくても、業務ルール、よくある質問、テンプレート、案件情報などを整理できます。そのため、非エンジニアがAI秘書に参照させる仕事の知識を蓄積する用途にも適しています。


### Markdownの基本的な書き方

Markdownでは、記号を使って文章の構造を表します。

---

📝 **Markdownの記載例**

```markdown
# 大見出し

## 小見出し

- 箇条書き1
- 箇条書き2

1. 手順1
2. 手順2

**重要な文章**

- [ ] 未完了の作業
- [x] 完了した作業

[リンクの表示名](https://example.com)

| 項目 | 内容 |
| --- | --- |
| 担当者 | 山田 |
| 期限 | 7月31日 |
```

---

見出し記号の `#` や箇条書きの `-` の後ろには、半角スペースを入れます。

### Markdownをプレビューする

VS Codeは、Markdownのプレビュー機能を標準で備えています。プレビューを使うと、見出しや箇条書きが実際にどのように表示されるか確認できます。

1. `.md` ファイルを開く
2. エディター右上の `プレビューを開く` アイコンを押す
3. 表示内容を確認する

| パソコン | プレビューを開く | 横にプレビューを開く |
| --- | --- | --- |
| macOS | `Shift + Command + V` | `Command + K` のあとに `V` |
| Windows、Linux | `Ctrl + Shift + V` | `Ctrl + K` のあとに `V` |

横にプレビューを開くと、左側で文章を編集しながら右側で表示結果を確認できます。

### 新しいファイルを作る

1. 左側のエクスプローラーで、保存先のフォルダーを右クリックする
2. `新しいファイル...` を選ぶ
3. ファイル名を入力する
4. `Enter` キーを押す
5. 中央のエディターに内容を入力する
6. ファイルを保存する

ファイル名の最後に `.md` を付けると、Markdownファイルとして認識されます。

### 新しいフォルダーを作る

1. 左側のエクスプローラーで、作成先のフォルダーを右クリックする
2. `新しいフォルダー...` を選ぶ
3. フォルダー名を入力する
4. `Enter` キーを押す

フォルダーやファイルを作成する前に、保存先が正しいことを確認してください。

### ファイル名を変更する

1. エクスプローラーで対象のファイルを右クリックする
2. `名前の変更...` を選ぶ
3. 新しいファイル名を入力する
4. `Enter` キーを押す

`.md` など、ファイル名の最後にある拡張子を誤って消さないように注意してください。

### ファイルを削除する

1. エクスプローラーで対象のファイルを右クリックする
2. `削除` を選ぶ
3. 削除するファイル名を確認する
4. 確認画面で削除を実行する

業務資料を削除する場合は、対象を間違えていないか必ず確認してください。

### ファイルを検索する

名前が分かっているファイルは、クイックオープンで探せます。

| パソコン | クイックオープン |
| --- | --- |
| macOS | `Command + P` |
| Windows、Linux | `Ctrl + P` |

1. クイックオープンを開く
2. ファイル名の一部を入力する
3. 候補から開きたいファイルを選ぶ

### フォルダー全体から文章を検索する

複数のファイルから、会社名やキーワードを探す場合に使います。

1. アクティビティ バーの `検索` アイコンを押す
2. 検索欄に探したい言葉を入力する
3. `Enter` キーを押す
4. 検索結果から該当箇所を選ぶ

| パソコン | フォルダー全体を検索 |
| --- | --- |
| macOS | `Command + Shift + F` |
| Windows、Linux | `Ctrl + Shift + F` |

置換機能を使うと複数ファイルの内容をまとめて変更できますが、意図しない箇所まで変わる可能性があります。最初は検索だけに使い、置換は変更対象を確認してから実行してください。
