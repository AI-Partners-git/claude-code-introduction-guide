# Claude CoworkでAI秘書を作ろう

**5万円で、一生使えるAI秘書をあなたのPCに誕生させる**

Claude Coworkを使って、自分・会社・プロジェクトの情報を理解し、必要なときに時間を問わず仕事を頼める「自分専用のAI秘書」を作るサービスです。

会社情報や仕事の進め方を追加・更新するほど、AI秘書はあなたのことを深く理解していきます。24時間365日いつでも頼むことができ、会社の情報や作業ルールを引き継ぎながら、辞めることなく使い続けられます。

## 人間の秘書とAI秘書の違い

| 比較項目 | 人間の秘書 | AI秘書 |
| --- | --- | --- |
| 費用 | 月20〜25万円（年250〜300万円） | 導入費用50,000円（税込） |
| 採用・教育 | 採用や教育に時間がかかる | 導入サービス内でセットアップ完了。あとは業務で使いながら成長させる |
| 稼働時間 | 勤務時間や休日がある | 必要なときに、24時間365日いつでも依頼できる |
| 継続性 | いつか辞める可能性がある | 退職することなく使い続けられる |
| あなた・会社への理解 | 入社後に少しずつ覚えてもらう | 登録した自分・会社・プロジェクトの情報をもとに、理解を深められる |

## サービス内容

- 料金：50,000円（税込）
- 合計時間：3〜5時間

| 内容 | 時間 | 行うこと |
| --- | --- | --- |
| Claude基礎 | 1〜2時間 | Claudeデスクトップの導入と初期設定、Skillへの作業ルールの登録、メモリへの会社情報やメール署名の登録、Gmailなどのコネクタ接続を行う |
| Claude Cowork基礎＋AI秘書セットアップ | 1〜2時間 | 業務フォルダを整理し、安全な権限を設定したうえで、自分の仕事に合わせたAI秘書を一緒に作る |
| 1週間後の定着確認MTG＋質問受付 | 1時間 | 実際に使って分かった疑問やつまずきを解消し、使い方や登録内容を修正する |

### 1週間後の定着確認が重要な理由

AI秘書は、作って終わりではありません。実際の仕事で使い、期待と違った部分を直しながら、自分に合う形へ育てていくことが大切です。

1週間後のMTGでは、使えなかった理由や迷った点を一緒に整理し、日常業務で使い続けられる状態に整えます。この確認が、AI活用を一度きりで終わらせず、習慣にするための重要な工程です。

## このサービスのゴール

**自分でAI秘書を「育て続けられる」状態にして卒業すること**

サービス終了後も、自分で会社情報や作業ルールを追加し、業務に合わせてAI秘書を改善できる状態を目指します。

## 導入後にできるようになること

- 「いつもの形で、株式会社ABCの佐藤様宛に営業メールを作って、Gmailに下書きしておいて」と一言で頼める
- 会社の型・署名・NG表現を反映した、そのまま送れる品質の文面を作れる
- 日報や商談メモをAI秘書に読ませて、週次の振り返りと来週のTODOを整理できる
- 商談前に資料を読み込ませ、論点整理とヒアリング質問リストを作れる

## こんな方におすすめ

- AIを使いたいが、何から始めればよいか分からない方
- ChatGPTやClaudeを触ったことはあるが、業務での使いどころが定まっていない方
- 自分や会社のことを理解したAI秘書に、日常業務を任せたい方
- 導入後も自分でAI秘書を育て、長く使い続けたい方

## 卒業後の任意追加サポート

以下は、5万円の導入サービスには含まれない有料の追加サポートです。導入サービスに含まれる「1週間後の定着確認MTG＋質問受付」とは別に、その後も相談したい場合のみ利用できます。

### フォローアップ・質問MTG（単発）

- 内容：オンラインMTG（30分）
- 料金：5,000円/回（税込）

導入サービス終了後の「ここでつまずいた」「この業務にも使えるか聞きたい」といった疑問を、必要なときに単発で相談できるMTGです。顧問契約を結ぶ前に、まずは気軽に相談したい方にもおすすめです。

#### 相談例

- AI秘書の設定の見直し・調整
- 操作方法やエラーなど、困りごとの解消
- 「この業務はAIでできるか？」という個別相談

### AI顧問プラン

AI顧問プランは、お客様の業務へのAI活用を継続的に進めるための任意のプランです。導入サービスで作ったAI秘書を入り口に、「どの業務をAIに任せられるか」を一緒に見つけ、実際に業務へ組み込むところまで伴走します。

| プラン | 料金 | 内容 |
| --- | --- | --- |
| チャットプラン | 5,000円/月 | チャットでいつでも質問・相談 |
| チャット＋月1回MTG | 15,000円/月 | チャット相談＋月1回のオンラインMTG |
| チャット＋週1回MTG | 30,000円/月 | チャット相談＋週1回のオンラインMTG |

※追加サポートの料金はすべて税込です。

#### チャットプランで行うこと（全プラン共通）

- 「この業務はAIでできるか？」という相談への回答と、具体的な進め方の提案
- 操作方法やエラーなど、困りごとへの随時回答
- 依頼文（プロンプト）の添削と改善提案
- Skill・メモリ・業務フォルダ構成の見直しアドバイス
- AIの新機能やアップデートのうち、お客様の業務に関係するものだけを厳選して共有

#### 月1回MTGプランで行うこと

チャットプランの内容に加えて、月1回のオンラインMTGで次を行います。

- 1ヶ月の活用状況の振り返りと、つまずきの解消
- 業務の棚卸しを行い、次にAI化する業務を一緒に決める
- 決めた業務のSkill・テンプレートの追加を一緒に実施

#### 週1回MTGプランで行うこと

月1回MTGプランの内容に加えて、週1回のペースで伴走します。

- 優先度の高い業務から順に、AIへの任せ方を毎週一緒に構築
- 業務フローの変更や新規案件への適用を、その週のうちに反映
- 社内の複数メンバーへの展開・定着の支援

#### プランの選び方の目安

- まずは困ったときに聞ける安心が欲しい → チャットプラン
- 毎月着実に、AIに任せる業務を増やしたい → チャット＋月1回MTG
- 短期間で一気に業務をAI化したい、社内に展開したい → チャット＋週1回MTG
