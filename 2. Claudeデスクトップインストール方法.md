## 2. Claudeデスクトップインストール方法

この章では、Claudeデスクトップアプリをパソコンに入れて、まずChatを使える状態にします。

### 事前に確認すること

インストール前に、Anthropicアカウントを用意してください。

Claudeデスクトップアプリ自体は無料プランでもインストールできます。

Chatを使うClaude基本編では、まず無料プランで問題ありません。

ただし、Coworkを使う場合は、Pro以上の有料プランが必要です。
一番お手頃なProプランで問題ありません。

CursorはClaudeとは別のアプリです。利用量や機能に制限はありますが、まずは無料プランで始められます。

| 使う範囲 | 必要なプランの目安 |
| --- | --- |
| Claude基本編：Chat、Skill、メモリ、コネクタの基本操作 | Claude無料プランで開始可能 |
| Cowork編：ローカルフォルダを使ってClaudeに作業を任せる | Claude Pro以上の有料プランが必要 |
| Cursor編：ファイルやフォルダを手動で編集する | Cursor無料プランで開始可能 |

※会社で使う場合は、すでにTeamやEnterpriseを契約している可能性があるため、先に社内の管理者へ確認してください。

- 料金ページ：https://claude.com/pricing

### インストール手順

1. Claudeのダウンロードページを開く

    - https://claude.com/download

2. 自分のOSに合うものを選ぶ

    | 利用しているパソコン | 選ぶもの |
    | --- | --- |
    | Mac | Download for macOS |
    | Windows | Download for Windows |
    | Windows ARM版 | Windows (arm64) |

    - ※Linux版のClaudeデスクトップアプリはbetaとして提供されています。Linux利用者は公式ドキュメントを確認してください。
    - [WindowsPCがX64かARM64か確認する方法](https://www.tku.ac.jp/iss/guide/classroom/soft/pcarmx64intelamd.html)


3. Anthropicアカウントでサインインする

    すでにClaudeをブラウザで使っている場合は、同じアカウントでサインインします。

    会社のTeamまたはEnterpriseアカウントを使う場合は、個人アカウントではなく会社で指定されたアカウントを選んでください。
