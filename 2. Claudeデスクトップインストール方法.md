## 2. Claudeデスクトップインストール方法

この章では、Claudeデスクトップアプリをパソコンに入れて、Claude Codeを使える状態にします。

### 事前に確認すること

インストール前に、次の2点を確認してください。

| 確認項目 | 内容 |
| --- | --- |
| アカウント | Anthropicアカウントが必要 |
| プラン | Claude Codeを使うには、Freeではなく対象プランが必要 |

Claudeデスクトップアプリ自体は無料プランでもインストールできます。ただし、Claude Codeを使う場合は、Pro、Max、Team、Enterprise、Consoleなどの対象プランをサブスク課金したアカウントが必要です。

一番お手頃なProプランで問題なく使用できます。

- ※会社で使う場合は、すでにTeamやEnterpriseを契約している可能性があるため、先に社内の管理者へ確認してください。

- 料金ページ：https://claude.com/pricing

### インストール手順

1. Claudeのダウンロードページを開く

    - https://claude.com/download

2. 自分のOSに合うものを選ぶ

    | 利用しているパソコン | 選ぶもの |
    | --- | --- |
    | Mac | Download for macOS |
    | Windows | Download for Windows |
    | Windows ARM版 | Windows (arm64) |

    - ※Linux版のClaudeデスクトップアプリは、公式ダウンロードページでは提供されていません。
    - [WindowsPCがX64かARM64か確認する方法](https://www.tku.ac.jp/iss/guide/classroom/soft/pcarmx64intelamd.html)


3. Anthropicアカウントでサインインする

    すでにClaudeをブラウザで使っている場合は、同じアカウントでサインインします。

    会社のTeamまたはEnterpriseアカウントを使う場合は、個人アカウントではなく会社で指定されたアカウントを選んでください。
