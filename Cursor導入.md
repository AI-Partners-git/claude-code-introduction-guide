## Cursor導入

### この章の位置づけ

ここまでの章では、AI秘書を育てて、まとまった仕事を任せられるようにしてきました。ただし、秘書が作った書類を最終確認し、細かい部分を自分の手で直すのは人間の仕事です。この章では、そのための道具を用意します。

Coworkは、Claudeに作業を依頼するには便利です。

一方で、フォルダやファイルを自分の手で直接開いて、文章を少し直したり、ファイル名を確認したり、Markdownを編集したりする用途には向いていません。

そのため、この章では AI秘書が作った書類の置き場所である `Claude業務フォルダ` を、人間が直接編集するための道具として、Cursorを使います。

Cursorは本来、開発者向けのAIエディタです。ただし、Markdownやテキストファイルを編集するだけなら、非エンジニアでも十分に使えます。

この章では、プログラム開発ではなく、次のような使い方に絞ります。

- `Claude業務フォルダ` を開く
- Markdownファイルを手で編集する
- ファイルの場所や中身を確認する
- 必要に応じて、CursorのAIに文章の整理を手伝わせる

Cursorの画面には、AIとの対話を中心にした `Agents`（エージェントモード）と、従来のエディタと同じ画面構成の `Editor`（エディターモード）の2つのレイアウトがあります。

| モード | 画面構成 | 向いている用途 |
| --- | --- | --- |
| エージェントモード | 左側にAIチャット、右側にファイル一覧 | AIに作業を任せて進める |
| エディターモード | 左側にファイル一覧、中央に編集画面 | 自分の手でファイルを開いて編集する |

この章の目的は、ファイルを自分の手で開いて編集することです。そのため、以降の手順はエディターモードを前提に進めます。

### 事前に確認すること

Cursorを入れる前に、次の点を確認してください。

| 確認項目 | 内容 |
| --- | --- |
| アカウント | Cursorの利用にはサインインが必要 |
| 対応OS | macOS、Windows、Linuxに対応 |
| Privacy Mode | 必要に応じて、設定からPrivacy Modeを有効にする |

Cursor公式のセキュリティページでは、Privacy ModeはFreeやProでも利用でき、有効にするとデータを学習に使わないと説明されています。

社外秘の資料、顧客情報、個人情報を扱う場合は、Cursorを使う前に会社のルールを確認してください。

### インストール手順

1. Cursorのダウンロードページを開く
    - https://cursor.com/download

2. 自分のOSに合うものを選ぶ
    - Mac
    - Windows
    - Linux

3. ダウンロードしたファイルを開き、画面の案内に従ってインストールする

4. Cursorを開き、アカウントでサインインする

5. Privacy Modeを有効にしておく

### エディターモードに切り替える

まず、画面がエディターモードになっているか確認します。

左側にファイル一覧、中央に編集画面が表示されていれば、すでにエディターモードです。

AIチャットが画面の中心に表示されている場合はエージェントモードになっているため、次の手順で切り替えます。

- 画面右上の`Editor Window`を押す

### Claude業務フォルダを開く

エディターモードにしたら、次の手順でフォルダを開きます。

1. Cursorを開く
2. 上部メニューの `File` を押す
3. `Open Folder` を選ぶ
4. `Claude業務フォルダ` を選ぶ
5. 左側にフォルダ一覧が表示されることを確認する
  - エディターモードでは、左側のファイル名をクリックすると、中央の編集画面で中身を開けます。

### ファイルを手で編集する

まずは、すでにあるMarkdownファイルを開いてみます。

1. 左側の `10_業務ルール` を開く
2. `営業メール作成ルール.md` を開く
3. 中央の編集画面で、必要があれば文章を少しだけ直す
4. 保存する

見た目を確認したい場合は、編集画面の右上にあるプレビューアイコンを押すと、Markdownの表示イメージを横に並べて確認できます。

保存のショートカットは次の通りです。

| パソコン | 保存 |
| --- | --- |
| Mac | `Command + S` |
| Windows | `Ctrl + S` |

### 新しいファイルを作る

次に、手で新しいMarkdownファイルを作ります。

1. 左側の `10_業務ルール` を右クリックする
2. `New File`を選ぶ
3. ファイル名を入力する

```text
社内共有メモ作成ルール.md
```

4. 次の内容を入力する

```text
# 社内共有メモ作成ルール

## 基本ルール

- 結論を最初に書く
- 決定事項と未決事項を分ける
- 次にやることは担当者と期限を書く
- 推測で事実を書かない
```

5. 保存する

このように、Cursorを使うと、Coworkに頼まなくても自分の手でファイルを追加、編集できます。

### CursorのAIに手伝わせる

CursorにはAI機能があります。

エディターモードでは、画面右側にAIチャットを表示できます。表示されていない場合は、次のショートカットで開きます。

| パソコン | AIチャットを開く |
| --- | --- |
| Mac | `Command + L` |
| Windows | `Ctrl + L` |

文章の整理を手伝わせる場合は、次のように依頼します。

```text
開いている社内共有メモ作成ルール.mdを確認して、
ルールの順番を分かりやすく並べ替える案を出してください。

まだファイルは編集しないでください。
まず提案だけ表示してください。
```

慣れてきたら、編集まで依頼できます。

```text
社内共有メモ作成ルール.mdを編集して、
ルールを「書き出し」「本文」「次のアクション」の3つに分類してください。

編集後、変更点を簡単に説明してください。
```
