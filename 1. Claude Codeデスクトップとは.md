## 1. Claude Codeデスクトップとは

### AIエージェントとは

AIエージェントとは、「この資料を直して」「このデータを集計して」「この内容で手順書を作って」のように、完成物を指定して仕事を依頼できるAIのことです。

ただし、ホテル検索や文章の要約だけであれば、最近のチャット型AIでもかなり対応できます。

Claude Codeデスクトップで違いが出るのは、パソコン内のフォルダや複数ファイルを使って、成果物の作成まで依頼する場面です。

依頼例：

> このフォルダに、営業資料、料金表、過去の提案書、議事録メモが入っています。  
> これらを読んで、中小企業の経営者向けに提案書の下書きを作ってください。  
> 古い料金や矛盾している説明があれば、別ファイルに一覧でまとめてください。

この場合、Claude Codeは次の順番で進めます。

1. フォルダ内のファイル一覧を確認する
2. 営業資料、料金表、過去の提案書、議事録メモを読む
3. 提案書に使えそうな内容を抜き出す
4. 料金表と提案書の内容が食い違っていないか確認する
5. 経営者向けに伝わりにくい表現を言い換える
6. 新しい提案書の下書きファイルを作る
7. 古い料金や確認が必要な箇所を、別ファイルに一覧化する

このように、単に回答を返すだけではなく、手元のファイルを読み、内容を照合し、新しいファイルとして成果物を残せる点がClaude Codeデスクトップの特徴です。

### チャット型AIとの違い

| 種類 | 得意なこと | 例 |
| --- | --- | --- |
| チャット型AI | その場で回答を返すこと | 「この文章を丁寧な表現に直して」「この資料を要約して」 |
| AIエージェント | 調査、整理、ファイル作成まで進めること | 「資料を読んで、比較表を作り、改善案を別ファイルにまとめて」 |

違いが出やすいのは、ファイルを扱う作業です。

チャット型AIでは、ユーザーがファイルをアップロードし、要約や文章作成を依頼する使い方が中心です。

Claude Codeデスクトップでは、作業用フォルダを指定して、その中のファイルをClaudeに読ませながら進められます。たとえば、以下のような依頼ができます。

> このフォルダに入っている営業資料を確認して、古い表現や修正した方がよい箇所を一覧にしてください。修正版のたたき台も作ってください。

この依頼に対して、Claudeはフォルダ内の資料を確認し、「古い表現の一覧」と「修正版の下書きファイル」を作成します。

### Claude Codeとは

Claude Codeは、Claudeがパソコン内のフォルダやファイルを確認しながら、資料の修正、データ整理、手順書作成などを行うための機能です。

名前に「Code」と入っていますが、できることはプログラミングだけではありません。非エンジニアの方でも、次のような業務で使えます。

| 業務 | Claude Codeへの依頼例 | 出てくる成果物 |
| --- | --- | --- |
| 手順書作成 | 「このメモをもとに、初心者向けの手順書に直してください」 | 見出しつきの手順書 |
| 営業資料の改善 | 「この提案書を読んで、経営者向けに伝わりにくい箇所を直してください」 | 修正案、改善ポイント一覧 |
| 議事録整理 | 「この議事録から決定事項、TODO、担当者を抜き出してください」 | TODO一覧、確認事項 |
| 表データの整理 | 「このCSVを見て、会社別・月別の売上を集計してください」 | 集計表、要約コメント |
| Webサイト修正 | 「このページの文言を、士業向けから中小企業向けに変更してください」 | 修正されたHTMLや文章案 |

Claude Codeが使いやすいのは、1つのファイルだけでなく、フォルダ全体を見ながら進められる点です。

たとえば、手順書を作る場合でも、`メモ.txt`、`既存資料.md`、`画面キャプチャ`、`参考URL一覧` などを同じフォルダに入れておけば、Claudeに「このフォルダを読んで手順書を作って」と依頼できます。

ただし、Claude Codeはファイルを読んだり編集したりできるため、最初は必ずコピーしたファイルで試してください。顧客情報、契約書、請求情報などを扱う場合は、社内ルールを確認してから使います。

### Chat、Cowork、Codeの違い

Claudeデスクトップでは、やりたい作業によって使う場所が分かれます。

| 機能 | 使う場面 | 具体例 |
| --- | --- | --- |
| Chat | すぐに相談したいとき | 「この文章を柔らかくして」「採用面談で聞く質問を考えて」 |
| Cowork | 資料作成や調査を1つの仕事として任せたいとき | 「競合3社を調べて、比較表と報告書を作って」 |
| Code | フォルダ内のファイルを読ませながら作業したいとき | 「このフォルダの下書きを直して、Markdown形式で整えて」 |

迷ったときは、次の基準で選びます。

- 会話だけで済むなら、Chatを使う
- 調査や資料作成を丸ごと任せたいなら、Coworkを使う
- 自分のパソコン内のフォルダやファイルを使って作業してほしいなら、Codeを使う

たとえば、同じ「営業資料を改善したい」という依頼でも、使い方は変わります。

| 依頼内容 | 向いている機能 |
| --- | --- |
| 「この営業資料の見出し案を10個出して」 | Chat |
| 「競合資料を調べて、自社資料の改善案をまとめて」 | Cowork |
| 「このフォルダにある営業資料を読み、表現を修正して別ファイルにして」 | Code |

本手順書では、非エンジニアの経営者や企業担当者でも実務で使えるように、Claude Codeデスクトップの基本操作、Skill、メモリ、コネクタ、具体的な活用例を順番に説明します。
